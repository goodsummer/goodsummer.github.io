hexo常用命令（持续更新）

发表于 2018-01-16 | 分类于 hexo相关

生成

hexo g or hexo generate

发布

hexo d or hexo deploy

发布与生成合并

hexo d -g

本地测试

hexo s or hexo server

默认地址：http://localhost:4000/， 端口可改。

清理本地缓存

hexo clean

新建文章

hexo new "我的文章"
